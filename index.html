プログラム復習コンテンツ

文字を描く

サンプルコードの結果

サンプルコード

　　 void setup(){

　　 　size(120, 120);　　

　　 　textSize(64);

　　 　textAlign(CENTER); 　

　　 }

　　

　　 void draw(){

　　 　background(0);

　　 　text(key, 60, 80);

　　 }

説明文

　関数textSize(), textAlign(), text(),

　　　　

　textSize()関数は、文字の大きさを設定する関数。

　textAlign()関数は、文字の揃えかたを設定する関数。

　　textAlign(align, valign)

　　　align　 水平方向の字揃え(LEFT、　CENTER、　RIGHT)の3種類。

　　　valign　垂直方向の字揃え(TOP、　CENTER、　BOTTOM、　BASELINE)の4種類。

　text()関数は、文字を表示する関数。

　　text(data, x, y)

　　　dataは表示する文字。xは表示する位置のx座標。yはy座標。

　

　例では、押したキーボードのボタンが表示される仕組みである。

　keyとはkey関数のことで、押したキーボードのボタンを値として算出する。
